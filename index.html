Authentication System (Backend)

AThis project is a secure authentication system built with Node.js, Express, and MongoDB. It uses twotypes of tokens for authentication: access tokens and refresh tokens. The access token is short-lived and used to authorize requests to protected routes, while the refresh token is long-lived and used to obtain a new access token when the previous one expires. The system also manages sessions and token invalidation to ensure robust security, with sensitive data protected using hashed tokens and HTTP-only cookies.
